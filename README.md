[http://resume-builder.surge.sh/](http://resume-builder.surge.sh/)


## Setup

To run this application, execute the following commands:

1. Install NPM modules

    ```
    $ npm install (or yarn install)
    ```

2. Start the front-end dev server:

    ```
    $ npm run start
    ```
