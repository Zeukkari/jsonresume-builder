Built with Grommet and JSON Resume.

## Development

Setup local development:

1. Clone the repo

   ```
   $ git clone https://github.com/Zeukkari/jsonresume-builder.git
   ```

2. Install dependencies

   ```
   $ npm install (or yarn install)
   ```

Start webpack dev server:

    ```
    $ npm start
    ```

Start Storybook:

    ```
    $ npm run storybook
    ```

Run tests:

    ```
    $ npm test
    ```
